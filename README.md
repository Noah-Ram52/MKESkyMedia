# MKE SkyMedia

MKE SkyMedia is a responsive web portfolio for a Milwaukee-based photography and videography brand. It showcases aerial drone work, ground-level visuals, an about section, contact information, and embedded media in a clean, modern layout designed to present professional creative services online.

## Design Layout

The homepage is structured to guide visitors through the brand story and contact information in a simple, visual flow:

- Header: The top navigation area introduces the brand name and provides quick access to the main sections, helping visitors move through the site easily.
- Hero/Main section: This is the first large visual area and is designed to immediately showcase the photography and videography style with strong imagery.
- About section: Located after the main visual content, this section explains who the brand is and what kind of work it offers, giving context to the portfolio.
- Contact section: Placed near the end of the page, this section provides social links, email, and phone details so visitors can reach out easily.
- Footer: The closing section reinforces branding and offers a polished finish to the page.

This layout works well because it leads the viewer from first impression to identity to action, keeping the experience focused and professional.
